# AI Study Assistant (Java)

This project is a Java-based AI Study Assistant that helps students generate smart study plans based on topic difficulty.

## Features
- Difficulty-based study planner
- Java Swing GUI
- Progress bar
- Save study plan to file

## Technologies Used
- Java
- Swing

## How it works
User enters topics with difficulty levels (easy, medium, hard).  
The system allocates time accordingly and generates a study plan.
